# LoanKaise.in Publishing System

## नया post कैसे publish करें

1. Repository में **Issues** खोलें।
2. **New issue** दबाएँ।
3. **Publish a new LoanKaise post** चुनें।
4. Post और SEO fields भरकर **Submit new issue** करें।
5. Workflow post, homepage, Loan Blog, category और sitemap update करके Cloudflare Pages पर deploy करेगा।

Live site: https://loankaise.in/

Publisher form: https://github.com/9097881689/loankaise-site/issues/new/choose
